# 🎨 Design System — Mpanolontsaina IA

Assistant juridique intelligent trilingue (malagasy / français), basé sur une architecture multi-agents (LangGraph + RAG ChromaDB + Vertex AI / Mistral).

---

## 1. Direction créative

**Mpanolontsaina** = "celui qui donne conseil" en malgache. L'app doit inspirer :
**confiance juridique + intelligence artificielle + identité malgache**, sans jamais tomber dans le froid/institutionnel.

| Principe | Traduction visuelle |
|---|---|
| Confiance & sérieux (juridique) | Violet profond, contrastes lisibles, hiérarchie claire |
| Intelligence / IA | Dégradés doux, halo subtil sur l'avatar IA, micro-animations "réflexion" |
| Chaleur malgache | Accent ocre/or, coins arrondis, ton conversationnel |
| Accessibilité trilingue | Sélecteur de langue toujours visible, jamais un texte en dur |

**Fichiers source uniques (ne jamais dupliquer les valeurs ailleurs) :**
- `src/theme/colors.js` → toutes les couleurs
- `src/i18n/translations.js` → tous les textes FR/MG

## 2. Palette de couleurs

Référence complète : `colors.js`. Toujours importer, jamais de hex en dur.

| Rôle | Token | Valeur |
|---|---|---|
| Marque principale | `colors.primary[600]` | `#5B2C9E` |
| Fond sombre / header | `colors.primary[900]` | `#1B1035` |
| Accent / CTA | `colors.accent[500]` | `#E2963C` |
| Texte principal | `colors.neutral[900]` | `#14121A` |
| Texte secondaire | `colors.neutral[600]` | `#524C63` |
| Fond page | `colors.neutral[50]` | `#FBFAFD` |
| Bulle IA | `colors.chat.aiBubbleBg` | `#F7F5FB` |
| Bulle utilisateur | `colors.chat.userBubbleBg` | `#5B2C9E` |
| Succès | `colors.semantic.success` | `#2FA86B` |
| Erreur | `colors.semantic.error` | `#E4483F` |

**Intégration React (2 options) :**

Option A — Tailwind (recommandé) :
```js
// tailwind.config.js
const { colors } = require("./src/theme/colors");
module.exports = { theme: { extend: { colors } } };
```
```jsx
<button className="bg-primary-600 text-white hover:bg-primary-700">Envoyer</button>
```

Option B — CSS Variables :
```js
// index.js (une fois au démarrage)
import { generateCSSVariables } from "./theme/colors";
const style = document.createElement("style");
style.innerHTML = generateCSSVariables();
document.head.appendChild(style);
```
```css
.button-primary { background: var(--color-primary-600); }
```

## 3. Typographie

| Usage | Police | Poids | Taille (desktop / mobile) |
|---|---|---|---|
| Titres (H1) | `Sora` (Google Fonts) — moderne, géométrique, forte lisibilité | 700 | 32px / 24px |
| Titres (H2/H3) | `Sora` | 600 | 22px / 18px |
| Corps de texte | `Inter` — excellente lisibilité multilingue (accents malgaches/français) | 400 | 16px / 15px |
| Bulles de chat | `Inter` | 400–500 | 15px / 14px |
| Boutons / labels | `Inter` | 600 | 14px |

```html
<link href="https://fonts.googleapis.com/css2?family=Sora:wght@600;700&family=Inter:wght@400;500;600&display=swap" rel="stylesheet">
```

> `Inter` gère nativement les diacritiques malgaches/français sans glyphes cassés — critère non négociable vu le trilinguisme.

---

## 4. Grille, espacement & rayons

- Base d'espacement : **4px** (4, 8, 12, 16, 24, 32, 48, 64)
- Rayon de bordure : `8px` (inputs/boutons), `16px` (cartes), `20px` (bulles de chat)
- Largeur max contenu : `1200px`, contenu chat : `760px` centré
- Breakpoints responsive :

| Nom | Largeur | Usage |
|---|---|---|
| `mobile` | < 640px | Chat plein écran, nav en bas |
| `tablet` | 640–1024px | Sidebar rétractable |
| `desktop` | > 1024px | Sidebar fixe + zone chat |

## 5. Composants clés

### 5.1 Bouton primaire
```jsx
<button className="px-5 py-3 rounded-lg bg-accent-500 text-neutral-900 font-semibold
  hover:bg-accent-600 active:scale-[0.98] transition disabled:opacity-40">
  {t("chat.send")}
</button>
```

### 5.2 Bulle de chat (utilisateur vs IA)
```jsx
<div className={isUser
  ? "bg-primary-600 text-white rounded-2xl rounded-br-sm px-4 py-3 ml-auto max-w-[80%]"
  : "bg-neutral-50 border border-neutral-200 text-neutral-900 rounded-2xl rounded-bl-sm px-4 py-3 max-w-[80%]"}>
  {content}
</div>
```

### 5.3 Indicateur "l'IA réfléchit" (3 points animés, couleur `chat.typingDot`)
### 5.4 Carte "source juridique" (article de loi cité) — fond `neutral-100`, bordure gauche `accent-500` 4px, icône livre
### 5.5 Sélecteur de langue MG/FR — toggle pill, toujours visible en haut à droite du header et dans la barre de chat
### 5.6 Input de chat — barre fixe en bas, bouton envoyer intégré, auto-resize textarea

## 6. États & feedback

| État | Traitement |
|---|---|
| Hover | Assombrissement 8–10% de la couleur de base |
| Focus (accessibilité) | Anneau `2px solid colors.primary[400]` + offset 2px |
| Disabled | Opacité 40%, curseur `not-allowed` |
| Chargement | Skeleton doux (`neutral-100` → `neutral-200` pulse) plutôt qu'un spinner générique |
| Erreur réseau/API | Bandeau `semantic.errorBg` + `semantic.error`, texte via `t("chat.errorGeneric")` |
| Rate limit visiteur (429) | Bandeau `semantic.warningBg`, incite à se connecter (`t("chat.rateLimited")`) |

---

## 7. Accessibilité (WCAG AA minimum)

- Contraste texte/fond ≥ 4.5:1 (vérifié : `neutral-900` sur `neutral-50` = 15.8:1 ✅)
- Toutes les icônes interactives ont un `aria-label` tiré de `translations.js`
- Navigation clavier complète dans le chat (Tab, Enter pour envoyer, Échap pour fermer les modales)
- `lang="fr"` ou `lang="mg"` sur la balise `<html>` mis à jour dynamiquement selon la langue active

---

## 8. Ton & micro-copy

- Toujours vouvoyer en français ("Veuillez...", jamais "Tu dois...")
- En malgache, registre respectueux standard (pas de argot)
- Les messages d'erreur restent factuels et rassurants, jamais alarmistes (contexte juridique sensible : licenciement, famille, foncier)
- L'IA se présente toujours comme un outil d'aide à l'information, jamais comme un avocat — mention discrète en pied de chat : "Mpanolontsaina IA fournit une information générale, non un avis juridique personnalisé."
